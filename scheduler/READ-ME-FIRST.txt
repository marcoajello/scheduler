🐛 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Nothing appears:
  → Hard refresh (Cmd/Ctrl + Shift + R)
  → Check browser console (F12)

  Drag not working:
  → Click the ⋮ handle specifically
  → Check JavaScript loaded

  Designer shows "No data":
  → Ensure you have schedule events
  → Refresh page
  → Check console for errors
